# git-br

git branch wrapper command

## Usage

```zsh
$ git br [-h|--help][-s|--sync]
```

## Installation

via zplug;

```zsh
zplug "jhawthorn/fzy", \
    as:command, \
    hook-build:'make'
zplug "b4b4r07/git-br", \
    as:command, \
    use:'git-br'
```
